# Car Parts France Pro - Site B2B pour Garages Professionnels

Ce projet est un site web React dédié aux garages professionnels, permettant à Car Parts France de présenter son service exclusif de fourniture de moteurs d'occasion testés et garantis.

## Fonctionnalités

- Présentation détaillée du service de moteurs d'occasion testés
- Description du processus de test rigoureux
- Explication des avantages pour les garages professionnels
- Formulaire de demande de devis personnalisé
- Témoignages clients
- Présentation du processus en 3 étapes
- Interface responsive adaptée à tous les appareils

## Technologies utilisées

- React 18
- TypeScript
- Material UI
- React Router
- Formik & Yup pour la gestion des formulaires
- Framer Motion pour les animations

## Installation

1. Cloner le dépôt
2. Installer les dépendances avec `npm install`
3. Lancer le serveur de développement avec `npm start`

## Structure du projet

- `/src/components/layout` - Composants de mise en page (Header, Footer)
- `/src/components/sections` - Sections de la page d'accueil
- `/src/pages` - Pages principales du site
- `/src/styles` - Styles et thème global
- `/public/images` - Images et ressources statiques

## Déploiement

Pour déployer le site en production :

```bash
npm run build
```

Cette commande génère une version optimisée du site dans le dossier `build`.
